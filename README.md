# Local Service Connect

A responsive service-booking website with complete Agile Scrum project management using Kanban board.

## Quick Links
- **Website:** http://localhost:8000/
- **Kanban Board:** Open in VS Code (Kanban extension by Marcel J. Kloubert)
- **Backlog File:** `kanban.json`

## Project Deliverables
✅ Frontend website (Home, About, Services, Booking, Contact)  
✅ Kanban board with 41 story points across 3 sprints  
✅ User stories and epics  
✅ Sprint planning documentation  
✅ Retrospectives and velocity tracking  

## How to Start

### 1. Run the Website
```bash
cd "c:\Users\kr360\Desktop\Agile project"
python -m http.server 8000
```
Then visit: http://localhost:8000/

### 2. Open Kanban Board in VS Code
- Install: Kanban board extension (Marcel J. Kloubert)
- Press Ctrl+Shift+P
- Type "Kanban"
- Open kanban.json file
